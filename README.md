# COVID Scaffold Filling Project

This project studies SARS-CoV-2 spike gene scaffold filling using deep learning methods.

## Files
- scaffoldDataset_prep.ipynb: data preprocessing and scaffold generation
- GPT2.ipynb: GPT-2 based training and prediction
- XiaoyinWanyan_Mid-term.pdf: mid-term report
- Presentation_midterm.pptx: project presentation
